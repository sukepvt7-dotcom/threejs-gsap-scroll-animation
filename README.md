# SOKKA

A production-ready Next.js App Router site for SOKKA, built with Tailwind CSS, GSAP, Three.js via React Three Fiber, and Lenis smooth scrolling.

## Stack

- Next.js
- Tailwind CSS
- GSAP + ScrollTrigger
- Three.js + React Three Fiber
- Lenis

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```
